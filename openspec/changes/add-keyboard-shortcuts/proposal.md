# Change: Add Keyboard Shortcuts

## Why

Power users need efficient keyboard shortcuts for:
- Faster workflow (no mouse required for common actions)
- Reduced fatigue during repetitive tasks
- Professional-grade tool feel

## What Changes

- Add global keyboard shortcut system
- Support common actions: save, undo, redo, generate, export
- Add shortcut hints in UI tooltips
- Allow users to customize shortcuts via settings

## Impact

- Affected specs: ui
- Affected code: src/components/Layout/, src/App.tsx
